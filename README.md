# AI 기반 세탁건조기 레이아웃 최적화 데모

RL+GNN & Diffusion+Transformer 기반 배치 최적화 시연 및 로드맵

## 페이지

- `/` — 랜딩 페이지
- `/tutorial.html` — RL & Diffusion 시연 튜토리얼 (폴리오미노 도형, Hill Climbing, 다중 패턴 Diffusion)
- `/roadmap.html` — 15개월 개발 로드맵 (시스템 아키텍처, Phase별 태스크, 마일스톤)

## 로컬 실행

```bash
python server.py
# http://localhost:8080 접속
```
