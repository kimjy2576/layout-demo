🧠RL + GNN 엔진 핵심
GNN: 부품 연결 그래프 인코더
노드 = 부품(크기, 위치, 스펙), 엣지 = 연결(냉매배관/덕트/하네스, 가중치). Edge-based GCN으로 각 부품의 "상황 인식" 임베딩 생성. 2~3 레이어 message passing.
PyTorch Geometric
EdgeGCN
RL: PPO 기반 순차 배치 에이전트
GNN 임베딩 + 캔버스 상태 → 정책 네트워크 → 격자 위 확률 분포 출력. 부품을 하나씩 순서대로 배치. PPO로 안정 학습. 전이 학습: 모델A 학습 → 모델B fine-tuning.

⚡멀티피직스 보상 시스템 핵심
열유체: 공기 사이클 + 냉동 사이클 통합 평가
배치 → 덕트 경로 → ductsim 압손 → 팬 운전점 → 풍량 결정 → HX-Sim 열교환 → compressor-sim 성능 → COP/건조시간 산출. HPWD 통합 사이클.
ductsim + HX-Sim + compressor-sim
진동: 1D 전달 경로 모델
컴프레서 가진 → 마운트 → 캐비닛 전달. 드럼 언밸런스 → 서스펜션 → 캐비닛. 질량-스프링-댐퍼 모델로 각 부품 위치별 진동 전달 계수 계산. 배관 고유진동수 간이 체크.
SciPy ODE
구조: 마운트 응력 + 운송 하중
컴포넌트 무게·위치에서 마운트 포인트 반력 계산. 운송 시 가속도 하중에서 각 마운트 응력 간이 계산. 임계값 초과 시 페널티.
세탁/건조 성능 종합 평가
건조 성능: dryer-drum-sim (건조시간, 에너지). 세탁 성능: 드럼 형상·위치 기반 세탁 효율 간이 모델. 소음: 진동+공력소음 합산. 이 모든 것을 가중 합산하여 하나의 보상 값 생성.
👨‍🔬전문가 검토 루프
제작/가공 가능성 검토 인터페이스
RL이 생성한 배치 후보 Top-5를 3D 뷰어로 전문가에게 제시. 사출 가능성, 조립 순서, 서비스 접근성, 배관 벤딩 가능성을 체크리스트로 평가. 전문가 점수가 보상에 반영 (Human-in-the-loop RLHF).

🌀Diffusion + Transformer 핵심
학습 데이터: RL 우수 배치 수집
Phase 2에서 RL+GNN이 발견한 물리 검증 완료 + 전문가 승인 배치를 학습 데이터로 축적. 다양한 캐비닛 스펙·부품 스펙 조건별 수천 건.
Diffusion Transformer (DiT) 디노이저 학습
U-Net 대신 Transformer 기반 디노이저 사용 (DiT 구조). 조건: 캐비닛 크기, 부품 스펙 리스트, 성능 목표. 조건부 생성으로 새 스펙 입력 시 수 초 내 배치 후보 다수 생성.
DiT (Diffusion Transformer)
PyTorch
Diffusion → RL Fine-tuning 파이프라인
Diffusion이 생성한 80점 배치를 RL이 95점으로 미세 조정. 이 결과를 다시 Diffusion 학습 데이터에 추가하는 선순환 루프.
🤖Surrogate AI (물리 모델 가속)
서브시스템별 Surrogate 신경망
덕트 Surrogate: 배치 좌표 → 압손 예측 (ms). 냉매 Surrogate: 배관 경로 → 압력강하 예측. 진동 Surrogate: 부품 위치 → 전달률 예측. Level 1 데이터 수천 건으로 MLP/GNN 학습.

sLLM(요구사항→스펙) → Diffusion(초기배치 생성) → RL(미세조정) → 멀티피직스(평가) → 전문가(승인) → 최종 배치. 전 과정 자동 오케스트레이션.

🔗 AI 기술별 역할 정리
sLLM — 사용자 소통(자연어→스펙), 1차 설계(부품 사이즈 결정), 설계 룰 자동 추출, 전문가 검토 보조
RL+GNN — 배치 탐색/최적화(보상 기반), 부품 연결 관계 이해, 새로운 배치 발견
Diffusion+Transformer — 우수 배치 빠른 대량 생성, 새 스펙 대응, RL 초기값 제공
Surrogate AI (MLP/GNN) — 물리 시뮬레이션 가속(ms 단위 평가), 모델 자동 보정
Bayesian Optimization — 스펙+배치 Co-design 상위 루프, 적은 횟수로 최적 탐색
LLM 텍스트 마이닝 — 과거 설계 이력/클레임에서 설계 룰 자동 추출, 물리 모델 문서화
🔗 물리 모델 역할 정리
열유체 — ductsim(압손), HX-Sim(열교환), compressor-sim(냉매사이클), fan-sim(PQ곡선) → COP, 건조시간
진동 — 질량-스프링-댐퍼 1D 모델 → 컴프레서/드럼 가진 전달률, 배관 고유진동수
구조 — 마운트 반력/응력 간이 계산, 운송 하중 체크
제어 — 인버터/팬/밸브 제어 최적화, quasi-steady 시간 영역
세탁/건조 성능 — dryer-drum-sim(건조), 세탁 효율 간이 모델, 소음 합산
